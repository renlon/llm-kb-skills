# Color Palette — MLL Knowledge Base

**Single source of truth for all diagram colors.** Edit this file to change the visual style of all generated diagrams.

---

## Shape Colors (Semantic)

| Semantic Purpose | Fill | Stroke | Use For |
|------------------|------|--------|---------|
| Concept | `#dbeafe` | `#1e40af` | Wiki concepts, definitions |
| Process | `#dcfce7` | `#166534` | Steps, actions, transformations |
| Data | `#ffedd5` | `#c2410c` | Inputs, outputs, data stores |
| Decision | `#fef3c7` | `#b45309` | Conditionals, branches |
| External | `#f3e8ff` | `#7c3aed` | External systems, APIs |
| Highlight | `#fecaca` | `#dc2626` | Warnings, key points, errors |

**Rule**: Always pair a darker stroke with a lighter fill for contrast.

---

## Text Colors (Hierarchy)

| Level | Color | Font Size | Use For |
|-------|-------|-----------|---------|
| Title | `#1e40af` | 28 | Diagram title |
| Heading | `#1e3a5f` | 20 | Section headings |
| Body | `#374151` | 16 | Standard labels, descriptions |
| Detail | `#64748b` | 14 | Annotations, metadata |
| On light fills | `#374151` | — | Text inside light shapes |
| On dark fills | `#ffffff` | — | Text inside dark shapes |

---

## Line & Arrow Colors

| Element | Color |
|---------|-------|
| Arrows | Stroke color of the source element |
| Structural lines | `#64748b` (slate) |
| Marker dots (fill + stroke) | `#3b82f6` (blue) |
